# PyBook
===PYTHON 2 NOT TESTED===

This is a notepad application that was made entirely in Python!

It includes such functions as:
- Saving
- Loading
- Dark Mode Switch
- Time and Date Input (Who Even Uses That)
- Font Changing
- Bold/Italic Text
- Changing Text Size
